# Hey there curious visitors!

I'm Omer, a 14-year old programmer from B&H.
If you came here, you're probably interested in me.
How did I begin programming?
It's a long and still incomplete story...

# When did I first hear about programming?

When I was 9, I saw those videos on YouTube, like "I coded a game in C from scratch" or "I just created my first fully-featured website" which got me thinking: how do people do it?
How to become a programmer?
Can I become one?

# My first "projects"

The very first programming language I went to learn was Lua.
Why?
Because I was using TIC-80.
Before that, I used Construct 3 to make games.
Some of them were good, some not as good, but I never published any, and I don't plan on doing so any time soon.
I learnt some Lua basics and tried making a game in TIC-80 (but never really succeeded).
Then, my father noticed my love for programming, so he assigned me for a course.

# My first time using Python

The course was online and we were learning the basics of Python.
Right off the bat I realized Python was very similar to Lua.
Or so I thought: turns out it's vice versa and Python is the "OG".
I ended up quitting TIC-80 (but I later came back, only to quit again and for good this time) and moving to Python which I'm using to this day.

# When I tried C++...and lost like a hundred brain cells

That one day, [this video](https://youtube.com/watch?v=VioxsWYzoJk) popped up on my YouTube "Recommended" tab.
It seemed interesting, so I decided to watch it.
To put it simply, it took Python around two minutes to count to one billion, while C++ did it in under three seconds.
I thought to myself: if Python is really that slow, won't the programs I make in Python be even slower?
Without thinking much I moved to C++...and instantly regreted it.
The first problem was that C++ is statically typed, and I was used to Python's duck typing.
Second, it was so hard to get used to having to place a semicolon after nearly every single statement.
However, I soon realized it just wasn't worth it, and switched back to Python after just a few weeks.

# My first school project

Since my principal (who really likes me) knew I was a programmer, he decided to ask me to create a buzzer system for him - like one of those on TV quizzes, where you have to press the buzzer first to answer the question.
First, I had almost no experience in web development at all.
Second, I knew it would take me at least a week to make it.
However, I went for it, as I felt like doing a challenge.
First, I had to learn CSS and expand my knowledge about HTML to be able to actually build the website.
When I showed it to the principal, he was immediately baffled - he definitely didn't expect me to have a solid concept in less than two days!
But little did I know the next five days would be nothing but pain.
I spent hours and hours learning about sockets and I had to learn another language: JavaScript.
Thankfully, JavaScript is literally a dynamically typed version of C++, so it didn't take long to get used to it.
I had to experiment for so long with the sockets and finally, one day before my deadline, I managed to finish it.
The website is in my language, Bosnian, but if you want me to post an English version, e-mail me - I still keep it on my hard drive.
Anyways, everything worked perfectly fine and it was also a great way of getting experience with web development.
However, I've still only scratched the surface, and I plan on learning more.

# My first competition

My teacher decided to assign me for a school competition in programming this year, but I hesitated at first because the language used in the competition was QBASIC, and I had no prior knowledge of QBASIC aside from what we learned during computer science classes.
We had intense preparations for a few weeks until, just a few days before the competition, we got great news: QBASIC was removed from the competition, and now you could only choose either Python or C++.
My two mates got scared at first, but knew it was okay when I showed them all my huge Python projects from the past few years.
The day came, and surely enough, one school immediately left because they prepared only QBASIC.
As it turns out, other schools started learning Python and C++ a month prior, so it was pure luck my team had me.
Anyways, we were first on town-level, which was s great start, but there is still a long way to go: there is state-level, country-level, and the olympics.
